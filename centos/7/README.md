# ansible-centos:7
Docker image with ansible based on the [official CentOS Docker Image](https://registry.hub.docker.com/_/centos/)

## Installed packages

Base:

- [CentOS (7) minimal](https://hub.docker.com/_/centos/)

Image specific:
- ansible

## Usage

```bash
$ docker run -it ansible-centos:7 /bin/bash
```
